HTML Questions:

What is HTML?

HTML stands for Hyper Text Markup Language.
It is used to create web pages.
It provides the structure and content of a webpage and its allowing browsers to display text, images, and other media .
HTML describes the structure of a Web page.
HTML consists of a series of elements.
HTML elements tell the browser how to display the content.

What are HTML tags?

HTML tags are used to tell the browser how to display a web page.
Eg:html tag,head tag,body tag.
